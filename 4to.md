# Что такое GIT
---
**GIT** это *базовый инутрсумент* для работы с кодом в команде. Во многих вакансия есть обязательное требование - ***навык работы с GIT***
**GIT** для новичка это прекрасный повод показать свое портфолио заказчику.


![](./asdasdff/developmentteam3.jpg)


---

***Как работает GIT***

**GIT** - консольная утилита,для *отслеживания и ведения истории изменения* файлов в проекте

При базовой работе с кодом, проект хранится на *локальном компьютере* и такой сценарий работы подходит для обучения.

В командной работе всегда присутвует GIT и при таком сценарии проект хранится на *удаленном сервере*. Там и находится репозиторий в котором хранятся папки и файлы проекта.

**Система контроля версий** позволяет организовать командную работу, когда каждый из разработчиков работает на своей локальной машине и затем обьединяют проекты в единую версию кода.
